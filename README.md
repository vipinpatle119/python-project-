# Python Calculator Project

This is a simple command-line calculator built using Python.

## Features
- Addition
- Subtraction
- Multiplication
- Division

## Technologies Used
- Python 3

## How to Run

1. Clone the repository

git clone https://github.com/your-username/python-project.git

2. Go to project folder

cd python-project

3. Run the program

python main.py
